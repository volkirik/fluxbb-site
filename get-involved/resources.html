FluxBB Documentation Contribute to the FluxBB Documentation using GitHub

FluxBB uses a GitHub repository to manage the official FluxBB Documentation, allowing FluxBB users and staff to easily collaborate and contribute to the documentation. To make contributions, you just need to fork the fluxbb-resources repository, add or update pages, push the changes to your forked repository, then send a pull request for review. Once the pull request is approved, your changes will be merged into the official FluxBB Documentation.

More information on making contributions to the documentation is available on the FluxBB Documentation repository on GitHub.
